Introduction to Swift

Swift is a powerful and intuitive programming language for iOS, macOS, watchOS, and tvOS development. It is designed to be easy to use, yet powerful enough for professional software development.

import UIKit

var greeting = "Hello, Swift!"
print(greeting)

Variables and Constants

In Swift, you declare constants using let and variables using var. Constants can't be changed once set, while variables can.

let city = "New York"
var temperature = 22

Functions

Functions in Swift allow you to bundle code that can be reused. Functions can have parameters, return values, and can even be passed as values themselves.

func greet(name: String) -> String {
    return "Hello, \(name)"
}

let message = greet(name: "Alex")

Object-Oriented Programming in Swift

Swift supports object-oriented programming concepts such as classes, inheritance, and polymorphism, allowing you to structure your code efficiently.

class Animal {
    var name: String
    init(name: String) {
        self.name = name
    }
}

class Dog: Animal {
    func bark() {
        print("Woof!")
    }
}

Optionals

Optionals are a powerful feature in Swift that allows variables to either have a value or be nil. This helps handle situations where a value might be absent.

var name: String? = "Alex"
if let unwrappedName = name {
    print("Hello, \(unwrappedName)")
}

Closures

Closures in Swift are self-contained blocks of functionality that can be passed around and used in your code. They are similar to lambdas in other languages.

let numbers = [1, 2, 3, 4, 5]
let doubled = numbers.map { $0 * 2 }

Collections

Swift provides powerful collection types like arrays, dictionaries, and sets, which are used to store and manipulate groups of related data.

let fruits = ["Apple", "Banana", "Cherry"]
let prices = ["Apple": 2.0, "Banana": 1.5]

Protocols

Protocols define methods and properties that can be adopted by any class, struct, or enum. They help in defining a blueprint for behaviors.

protocol Driveable {
    func drive()
}

class Car: Driveable {
    func drive() {
        print("Driving the car")
    }
}

Advanced Swift

Advanced topics in Swift include memory management, concurrency, and performance optimization. These concepts help you create more efficient and scalable applications.

struct Temperature {
    var celsius: Double
    var fahrenheit: Double {
