Introduction to Kotlin

Kotlin is a modern, statically-typed programming language that runs on the JVM, Android, JavaScript, and native platforms. Officially supported by Google for Android development.

fun main() {
    println("Hello, Kotlin!")
}

Basic Syntax

Key Features:

  • val - Immutable variables
  • var - Mutable variables
  • Type inference
val name = "Alex"  // String inference
var age: Int = 25  // Explicit type
val isAdult = age >= 18  // Boolean

Null Safety

Kotlin's type system eliminates null pointer exceptions:

var safeString: String? = null  // Nullable type
val length = safeString?.length ?: 0  // Elvis operator

safeString!!.length  // Assert non-null (risky)

Functions

// Single-expression function
fun square(x: Int) = x * x

// Default arguments
fun greet(name: String = "Guest") {
    println("Hello, $name!")
}

OOP in Kotlin

open class Animal(val name: String) {
    open fun sound() = "Generic animal sound"
}

class Cat(name: String) : Animal(name) {
    override fun sound() = "Meow!"
}

interface Swimmable {
    fun swim()
}

Data Classes

data class User(
    val id: Int,
    val name: String,
    val email: String
)

val user = User(1, "Alex", "alex@example.com")
val copy = user.copy(email = "new@example.com")

Extension Functions

fun String.addExclamation() = "$this!"

val greeting = "Hello".addExclamation()  // "Hello!"

Lambda Expressions

val numbers = listOf(1, 2, 3)
val doubled = numbers.map { it * 2 }

// Higher-order functions
fun repeat(times: Int, action: (Int) -> Unit) {
    for (i in 1..times) action(i)
}

Collections

val immutableList = listOf("A", "B", "C")
val mutableMap = mutableMapOf(1 to "One", 2 to "Two")

val filtered = immutableList.filter { it.startsWith("A") }
