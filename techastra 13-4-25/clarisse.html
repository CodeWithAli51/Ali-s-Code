Introduction to PHP

PHP is a server-side scripting language used for web development. It is embedded in HTML and is widely used to create dynamic web pages.

<?php
echo "Hello, World!";
?>

PHP Syntax

PHP scripts start with <?php and end with ?>. PHP statements must end with a semicolon (;).

<?php
echo "This is PHP syntax!";
?>

Variables and Data Types

Variables in PHP start with a dollar sign ($). PHP supports various data types like strings, integers, floats, booleans, arrays, and objects.

<?php
$name = "John";
$age = 25;
$isStudent = true;
?>

Loops

PHP supports loops like for, while, and foreach.

<?php
for ($i = 0; $i < 5; $i++) {
    echo "Number: $i<br>";
}
?>

Functions

Functions in PHP are defined using the function keyword.

<?php
function greet($name) {
    return "Hello, $name!";
}
echo greet("Alice");
?>

Arrays

Arrays in PHP can store multiple values in a single variable.

<?php
$colors = array("Red", "Green", "Blue");
echo $colors[0]; // Output: Red
?>

Working with Forms

PHP can handle form data using the $_POST or $_GET superglobals.

<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $name = $_POST["name"];
    echo "Hello, $name!";
}
?>

File Handling

PHP can read and write files using functions like fopen, fread, and fwrite.

<?php
$file = fopen("example.txt", "w");
fwrite($file, "Hello, World!");
fclose($file);
?>

Database Connectivity

PHP can connect to databases like MySQL using the mysqli extension.

<?php
$conn = new mysqli("localhost", "root", "", "mydb");
if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
}
echo "Connected successfully";
?>

Error Handling

PHP provides functions like try, catch, and throw for error handling.

<?php
try {
    // Code that may throw an exception
} catch (Exception $e) {
    echo "Error: " . $e->getMessage();
}
?>

Introduction to Ruby

Ruby is a dynamic, object-oriented programming language known for its simplicity and productivity.

puts "Hello, World!"

Ruby Syntax

Ruby code is easy to read and write. It uses keywords like def for defining methods and end to close blocks.

def greet(name)
    puts "Hello, #{name}!"
end
greet("Alice")

Variables and Data Types

Ruby supports variables and data types like strings, integers, floats, and booleans.

name = "John"
age = 25
is_student = true

Loops

Ruby supports loops like for, while, and each.

for i in 0..4
    puts "Number: #{i}"
end

Methods

Methods in Ruby are defined using the def keyword.

def add(a, b)
    return a + b
end
puts add(2, 3) # Output: 5

Classes and Objects

Ruby is an object-oriented language. Classes are defined using the class keyword.

class Person
    def initialize(name)
        @name = name
    end
    def greet
        puts "Hello, #{@name}!"
    end
end
person = Person.new("Alice")
person.greet

Modules

Modules in Ruby are used to group methods, classes, and constants.

module Greet
    def say_hello
        puts "Hello!"
    end
end
class Person
    include Greet
end
person = Person.new
person.say_hello

Exception Handling

Ruby uses begin, rescue, and ensure for exception handling.

begin
    # Code that may raise an exception
rescue => e
    puts "Error: #{e.message}"
end

Introduction to SQL

SQL (Structured Query Language) is used to manage and manipulate relational databases.

SELECT * FROM users;

SQL Syntax

SQL statements are used to perform tasks like querying data, updating data, and managing databases.

SELECT column1, column2 FROM table_name WHERE condition;

SELECT Queries

The SELECT statement is used to retrieve data from a database.

SELECT name, email FROM users WHERE age > 18;

INSERT, UPDATE, DELETE

SQL supports commands like INSERT, UPDATE, and DELETE to modify data.

INSERT INTO users (name, email) VALUES ('John', 'john@example.com');
UPDATE users SET email = 'john.doe@example.com' WHERE id = 1;
DELETE FROM users WHERE id = 1;

Joins

SQL joins are used to combine rows from two or more tables.

SELECT users.name, orders.amount
FROM users
INNER JOIN orders ON users.id = orders.user_id;

Aggregation Functions

SQL provides functions like COUNT, SUM, AVG, MIN, and MAX.

SELECT COUNT(*) FROM users;
SELECT AVG(age) FROM users;

Subqueries

Subqueries are queries nested inside another query.

SELECT name FROM users WHERE age > (SELECT AVG(age) FROM users);

Indexes and Optimization

Indexes are used to speed up database queries.

CREATE INDEX idx_name ON users (name);

Introduction to MySQL

MySQL is an open-source relational database management system (RDBMS) that uses SQL.

-- Example: Create a table
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100)
);

Installation

MySQL can be installed on various operating systems. Follow the official documentation for installation steps.

Creating Databases

Use the CREATE DATABASE statement to create a new database.

CREATE DATABASE mydb;

Creating Tables

Use the CREATE TABLE statement to create a new table.

CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100)
);

CRUD Operations

CRUD stands for Create, Read, Update, and Delete. These are the basic operations performed on a database.

-- Create
INSERT INTO users (name, email) VALUES ('John', 'john@example.com');

-- Read
SELECT * FROM users;

-- Update
UPDATE users SET email = 'john.doe@example.com' WHERE id = 1;

-- Delete
DELETE FROM users WHERE id = 1;

Joins

MySQL supports various types of joins like INNER JOIN, LEFT JOIN, and RIGHT JOIN.

SELECT users.name, orders.amount
FROM users
INNER JOIN orders ON users.id = orders.user_id;

Transactions

Transactions ensure that a set of SQL operations are executed as a single unit.

START TRANSACTION;
INSERT INTO users (name, email) VALUES ('Alice', 'alice@example.com');
INSERT INTO orders (user_id, amount) VALUES (LAST_INSERT_ID(), 100);
COMMIT;

Backup and Restore

MySQL provides tools like mysqldump for backup and mysql for restoring databases.

# Backup
mysqldump -u root -p mydb > mydb_backup.sql

# Restore
mysql -u root -p mydb < mydb_backup.sql

Introduction to Oracle

Oracle is a powerful RDBMS used for enterprise-level applications. It supports complex queries and large datasets.

-- Example: Create a table
CREATE TABLE employees (
    employee_id NUMBER PRIMARY KEY,
    first_name VARCHAR2(50),
    last_name VARCHAR2(50)
);

Installation

Oracle can be installed on various operating systems. Follow the official documentation for installation steps.

Creating Databases

Use the CREATE DATABASE statement to create a new database.

CREATE DATABASE mydb;

Creating Tables

Use the CREATE TABLE statement to create a new table.

CREATE TABLE employees (
    employee_id NUMBER PRIMARY KEY,
    first_name VARCHAR2(50),
    last_name VARCHAR2(50)
);

PL/SQL Basics

PL/SQL is Oracle's procedural extension to SQL. It allows you to write procedural code like loops and conditions.

DECLARE
    name VARCHAR2(50);
BEGIN
    name := 'John';
    DBMS_OUTPUT.PUT_LINE('Hello, ' || name);
END;

Joins

Oracle supports various types of joins like INNER JOIN, LEFT JOIN, and RIGHT JOIN.

SELECT employees.first_name, departments.department_name
FROM employees
INNER JOIN departments ON employees.department_id = departments.department_id;

Transactions

Transactions ensure that a set of SQL operations are executed as a single unit.

BEGIN
    INSERT INTO employees (employee_id, first_name, last_name) VALUES (1, 'John', 'Doe');
    INSERT INTO departments (department_id, department_name) VALUES (10, 'Sales');
    COMMIT;
END;

Backup and Restore

Oracle provides tools like RMAN for backup and recovery.

# Backup
RMAN> BACKUP DATABASE;

# Restore
RMAN> RESTORE DATABASE;

Introduction to Cisco Packet Tracer

Cisco Packet Tracer is a network simulation tool used to design, configure, and troubleshoot network systems.

Router> enable
Router# configure terminal
Router(config)# hostname R1
R1(config)# interface gigabitethernet 0/0
R1(config-if)# ip address 192.168.1.1 255.255.255.0
R1(config-if)# no shutdown

Installation

Cisco Packet Tracer can be downloaded from the official Cisco Networking Academy website.

Basic Configuration

Basic configuration includes setting up IP addresses, hostnames, and enabling interfaces.

Router> enable
Router# configure terminal
Router(config)# hostname R1
R1(config)# interface gigabitethernet 0/0
R1(config-if)# ip address 192.168.1.1 255.255.255.0
R1(config-if)# no shutdown

Routing

Routing involves configuring routers to forward packets between networks.

R1(config)# ip route 192.168.2.0 255.255.255.0 192.168.1.2

Switching

Switching involves configuring switches to forward frames within a network.

Switch> enable
Switch# configure terminal
Switch(config)# vlan 10
Switch(config-vlan)# name Sales

Troubleshooting

Troubleshooting involves identifying and resolving network issues using tools like ping and traceroute.

Router# ping 192.168.1.1
Router# traceroute 192.168.2.1

Introduction to C#

C# is a modern, object-oriented programming language developed by Microsoft. It is widely used for building Windows applications and games.

using System;

class Program {
    static void Main() {
        Console.WriteLine("Hello, World!");
    }
}

C# Syntax

C# code is structured using namespaces, classes, and methods.

using System;

namespace HelloWorld {
    class Program {
        static void Main(string[] args) {
            Console.WriteLine("Hello, World!");
        }
    }
}

Variables and Data Types

C# supports variables and data types like int, string, bool, and double.

int age = 25;
string name = "John";
bool isStudent = true;
double salary = 50000.50;

Loops

C# supports loops like for, while, and foreach.

for (int i = 0; i < 5; i++) {
    Console.WriteLine("Number: " + i);
}

Functions

Functions in C# are defined using the void or return keyword.

int Add(int a, int b) {
    return a + b;
}
Console.WriteLine(Add(2, 3)); // Output: 5

OOP Concepts

C# supports object-oriented programming concepts like classes, objects, inheritance, and polymorphism.

class Person {
    public string Name { get; set; }
    public void Greet() {
        Console.WriteLine("Hello, " + Name);
    }
}
Person person = new Person();
person.Name = "Alice";
person.Greet();

Exception Handling

C# uses try, catch, and finally for exception handling.

try {
    int result = 10 / 0;
} catch (DivideByZeroException e) {
    Console.WriteLine("Error: " + e.Message);
}

File Handling

C# can read and write files using the System.IO namespace.

using System.IO;

string path = "example.txt";
File.WriteAllText(path, "Hello, World!");
string content = File.ReadAllText(path);
Console.WriteLine(content);
