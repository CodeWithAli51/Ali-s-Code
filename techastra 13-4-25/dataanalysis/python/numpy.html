NumPy: Numerical Computing with Python

NumPy (Numerical Python) is a fundamental library for scientific computing in Python. It provides support for efficient multidimensional arrays, mathematical functions, and broadcasting.

Why Use NumPy?

  • Efficient Arrays: NumPy arrays are faster and more memory-efficient than Python lists.
  • Mathematical Functions: Built-in operations for linear algebra, statistics, and Fourier transforms.
  • Broadcasting: Perform element-wise operations on arrays of different shapes.
  • Integration: Works seamlessly with other libraries like Pandas, Matplotlib, and Scikit-learn.

Installation of NumPy

If you haven't installed NumPy yet, use pip:


                    pip install numpy

To confirm that NumPy is installed correctly, run the following in Python:


                    import numpy as np
                    print(np.__version__)

Key Features of NumPy

1. Arrays

A NumPy array is a grid of values, all of the same type, indexed by a tuple of non-negative integers.

Creating Arrays

                    import numpy as np
                    
                    # Create an array from a list
                    arr = np.array([1, 2, 3, 4, 5])
                    print(arr)
                    
                    # Create a 2D array
                    arr_2d = np.array([[1, 2, 3], [4, 5, 6]])
                    print(arr_2d)
Accessing Elements

                    print(arr[0])  # Output: 1
                    print(arr_2d[1, 2])  # Output: 6
Array Attributes

                    print(arr.shape)  # Shape of the array
                    print(arr.dtype)  # Data type of elements
                    print(arr.ndim)   # Number of dimensions

2. Array Operations

NumPy allows you to perform element-wise operations efficiently.

Arithmetic Operations

                    arr1 = np.array([1, 2, 3])
                    arr2 = np.array([4, 5, 6])
                    
                    # Element-wise addition
                    print(arr1 + arr2)  # Output: [5, 7, 9]
                    
                    # Element-wise multiplication
                    print(arr1 * arr2)  # Output: [4, 10, 18]
Scalar Operations

                    print(arr1 + 10)  # Add 10 to each element
                    print(arr1 * 2)   # Multiply each element by 2

3. Broadcasting

Broadcasting allows NumPy to perform operations on arrays of different shapes.


                    arr = np.array([1, 2, 3])
                    scalar = 10
                    
                    # Broadcasting scalar to array
                    print(arr + scalar)  # Output: [11, 12, 13]

Common NumPy Functions

1. Mathematical Functions


                    arr = np.array([1, 2, 3, 4, 5])
                    
                    # Mean
                    print(np.mean(arr))  # Output: 3.0
                    
                    # Median
                    print(np.median(arr))  # Output: 3.0
                    
                    # Standard Deviation
                    print(np.std(arr))  # Output: 1.414...

2. Trigonometric Functions


                    angles = np.array([0, np.pi / 2, np.pi])
                    
                    # Sine
                    print(np.sin(angles))
                    
                    # Cosine
                    print(np.cos(angles))

3. Linear Algebra


                    matrix1 = np.array([[1, 2], [3, 4]])
                    matrix2 = np.array([[5, 6], [7, 8]])
                    
                    # Matrix multiplication
                    result = np.dot(matrix1, matrix2)
                    print(result)
                    
                    # Transpose
                    print(matrix1.T)  # Transpose of matrix1

4. Random Numbers


                    # Random integers
                    random_ints = np.random.randint(0, 10, size=5)  # 5 random integers between 0 and 10
                    print(random_ints)
                    
                    # Random floats
                    random_floats = np.random.rand(3, 3)  # 3x3 array of random floats between 0 and 1
                    print(random_floats)

Practical Example: Analyzing a Dataset


                    data = np.array([
                        [10, 20, 30],
                        [40, 50, 60],
                        [70, 80, 90]
                    ])
                    
                    # Calculate row and column means
                    row_means = np.mean(data, axis=1)  # Mean of each row
                    col_means = np.mean(data, axis=0)  # Mean of each column
                    print("Row Means:", row_means)
                    print("Column Means:", col_means)
                    
                    # Normalize the data
                    normalized_data = (data - np.mean(data)) / np.std(data)
                    print("Normalized Data:")
                    print(normalized_data)
                    
                    # Filter values
                    filtered_data = data[data > 50]  # Select values greater than 50
                    print("Filtered Data:", filtered_data)
