Pandas: Data Manipulation and Analysis

Pandas is an open-source Python library specifically designed for data manipulation and analysis. It provides powerful tools for working with structured data (e.g., tabular data like Excel sheets or SQL tables) and is widely used in data science, machine learning, and analytics.

Why Use Pandas?

  • Data Structures: Pandas introduces two primary data structures:
    • Series: A one-dimensional labeled array.
    • DataFrame: A two-dimensional labeled data structure with rows and columns.
  • Data Cleaning: Simplifies handling missing data, duplicate rows, and inconsistent formats.
  • Data Transformation: Supports operations like filtering, grouping, merging, and reshaping.
  • Integration: Works seamlessly with other libraries like NumPy, Matplotlib, and Scikit-learn.
  • Performance: Optimized for performance, making it suitable for large datasets.

Installation of Pandas

If you haven't installed Pandas yet, use pip:


                pip install pandas

To confirm that Pandas is installed correctly, run the following in Python:


                import pandas as pd
                print(pd.__version__)

Key Features of Pandas

1. Series

A Series is a one-dimensional array-like object containing data and labels (indices).

Creating a Series

                import pandas as pd
                
                # Create a Series from a list
                data = [10, 20, 30, 40]
                series = pd.Series(data)
                print(series)
Accessing Elements

                print(series[0])  # Output: 10
Custom Indices

                series = pd.Series(data, index=['a', 'b', 'c', 'd'])
                print(series['b'])  # Output: 20

2. DataFrame

A DataFrame is a two-dimensional table-like structure with rows and columns. It is the most commonly used data structure in Pandas.

Creating a DataFrame

                # Create a DataFrame from a dictionary
                data = {
                    'Name': ['Alice', 'Bob', 'Charlie'],
                    'Age': [25, 30, 35],
                    'City': ['New York', 'Los Angeles', 'Chicago']
                }
                df = pd.DataFrame(data)
                print(df)
Accessing Columns

                print(df['Name'])  # Access a single column
                print(df[['Name', 'Age']])  # Access multiple columns
Accessing Rows

                print(df.iloc[0])  # Access the first row by index
                print(df.loc[0])  # Access the first row by label

Data Cleaning with Pandas

Data cleaning is the process of identifying and correcting errors or inconsistencies in datasets to make them suitable for analysis.

Common Data Cleaning Tasks

1. Handling Missing Data

                # Drop missing values
                df.dropna(inplace=True)
                
                # Fill missing values
                df['Age'].fillna(df['Age'].mean(), inplace=True)
2. Removing Duplicates

                df.drop_duplicates(inplace=True)
3. Converting Data Types

                df['Age'] = df['Age'].astype(float)
4. Renaming Columns

                df.rename(columns={'OldName': 'NewName'}, inplace=True)

Data Transformation with Pandas

1. Filtering Data


                filtered_df = df[df['Age'] > 30]  # Select rows where Age > 30
                print(filtered_df)

2. Adding New Columns


                df['Salary'] = [50000, 60000, 70000]  # Add a new column
                print(df)

3. Grouping and Aggregation


                grouped = df.groupby('City')['Age'].mean()  # Average age per city
                print(grouped)

4. Merging DataFrames


                df1 = pd.DataFrame({'Key': ['A', 'B'], 'Value': [1, 2]})
                df2 = pd.DataFrame({'Key': ['A', 'C'], 'Value': [3, 4]})
                merged = pd.merge(df1, df2, on='Key', how='inner')  # Inner join
                print(merged)

Exploratory Data Analysis (EDA) with Pandas

1. Descriptive Statistics


                print(df.describe())

2. Value Counts


                print(df['City'].value_counts())

3. Sorting Data


                sorted_df = df.sort_values(by='Age', ascending=False)
                print(sorted_df)

Advanced Features of Pandas

1. Pivot Tables


                pivot = df.pivot_table(values='Age', index='City', aggfunc='mean')
                print(pivot)

2. Time Series Analysis


                df['Date'] = pd.date_range(start='2023-01-01', periods=len(df))
                print(df)

3. Reading and Writing Files


                # Read CSV
                df = pd.read_csv('data.csv')
                
                # Write to CSV
                df.to_csv('output.csv', index=False)

Practical Example: Analyzing a Dataset


                data = {
                    'Name': ['Alice', 'Bob', 'Charlie', 'David'],
                    'Age': [25, 30, 35, None],
                    'City': ['New York', 'Los Angeles', 'Chicago', 'Houston'],
                    'Salary': [50000, 60000, 70000, 80000]
                }
                df = pd.DataFrame(data)
                
                # Handle Missing Data
                df['Age'].fillna(df['Age'].mean(), inplace=True)
                
                # Filter High Earners
                high_earners = df[df['Salary'] > 60000]
                print(high_earners)
                
                # Group by City
                grouped = df.groupby('City')['Salary'].mean()
                print(grouped)
                
                # Visualize Data
                import matplotlib.pyplot as plt
                df['Salary'].plot(kind='bar')
                plt.show()
