Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page and consists of a series of elements.

Key Features:

  • Structure: HTML defines the structure of a web page.
  • Elements: HTML elements are the building blocks of a web page.
  • Tags: HTML uses tags to define elements.
  • Cross-Platform: HTML works on all browsers and devices.
<!DOCTYPE html>
<html>
<head>
    <title>My First Web Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
</body>
</html>

HTML Basics

HTML documents are made up of elements, which are defined by tags. Tags are enclosed in angle brackets (< >).

Basic Structure:

  • DOCTYPE: Declares the document type and version of HTML.
  • html: The root element of an HTML document.
  • head: Contains meta-information about the document.
  • body: Contains the content of the document.
<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
</body>
</html>

HTML Elements

HTML elements are the building blocks of a web page. They consist of a start tag, content, and an end tag.

Example:

<p>This is a paragraph.</p>

Common Elements:

  • <h1> to <h6>: Headings.
  • <p>: Paragraphs.
  • <a>: Links.
  • <img>: Images.

HTML Attributes

Attributes provide additional information about HTML elements. They are always specified in the start tag.

Common Attributes:

  • id: Specifies a unique ID for an element.
  • class: Specifies one or more class names for an element.
  • src: Specifies the source URL for an image.
  • href: Specifies the URL for a link.
<a href="https://example.com">Visit Example</a>

Headings

Headings are used to define the structure of a web page. There are six levels of headings, from <h1> to <h6>.

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>

Paragraphs

Paragraphs are used to group text content. They are defined using the <p> tag.

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

Links

Links are used to navigate between web pages. They are defined using the <a> tag.

<a href="https://example.com">Visit Example</a>

Images

Images are used to display pictures on a web page. They are defined using the <img> tag.

<img src="image.jpg" alt="Description of image">

Lists

Lists are used to group related items. There are two types of lists: ordered (<ol>) and unordered (<ul>).

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

Tables

Tables are used to display data in rows and columns. They are defined using the <table> tag.

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
    </tr>
</table>

Forms

Forms are used to collect user input. They are defined using the <form> tag.

<form action="/submit" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <input type="submit" value="Submit">
</form>

Semantic HTML

Semantic HTML uses meaningful tags to describe the content of a web page. Examples include <header>, <footer>, and <article>.

<header>
    <h1>Website Title</h1>
</header>
<article>
    <p>This is an article.</p>
</article>
<footer>
    <p>Footer content</p>
</footer>

Multimedia

HTML supports multimedia elements like audio and video. They are defined using the <audio> and <video> tags.

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
</audio>

<video controls>
    <source src="video.mp4" type="video/mp4">
</video>

Accessibility

Accessibility ensures that web pages are usable by everyone, including people with disabilities. Use semantic HTML and ARIA attributes to improve accessibility.

<button aria-label="Close">X</button>

Best Practices

Follow these best practices to write clean and maintainable HTML:

  • Use semantic HTML.
  • Always include the alt attribute for images.
  • Use lowercase for tags and attributes.
  • Close all tags properly.
