Introduction to CSS

CSS (Cascading Style Sheets) is the language used to style and format web pages. It defines how HTML elements appear on the screen, including layout, colors, fonts, and animations. Mastering CSS enables developers to create visually appealing and responsive designs.

Example: <!DOCTYPE html>
                <html>
                <head>
                    <style>
                        p {
                            color: blue;
                            font-size: 20px;
                        }
                    </style>
                </head>
                <body>
                    <p>Hello, CSS!</p>
                </body>
                </html>

Ways to Apply CSS

CSS can be implemented using three methods:

1 Inline CSS

  • Written inside an HTML element using the style attribute.
  • Example: (style="color: blue;">Hello)
  • Used for quick fixes but not recommended for large projects.

2. Internal CSS

  • Written inside the (style) tag in the (head) section of an HTML file.
  • Useful for single-page styling but difficult to manage across multiple pages.

3. External CSS

  • Stored in a separate .css file and linked using (link rel="stylesheet" href="style.css").
  • Best practice for maintaining consistency and reusability across multiple web pages.
Example 1: <p style="color: blue;">Hello</p>
                
Example 2: <head>
                <style>
                    p {
                        color: blue;
                    }
                </style>
            </head>
            <body>
                <p>Hello</p>
            </body>
            
Example 3: <head>
            <link rel="stylesheet" href="style.css">
        </head>

Selectors in CSS

Selectors are used to target and style specific HTML elements.

1. Basic Selectors

  • Universal Selector (*): Styles all elements.
  • Element Selector (h1, p): Targets specific HTML tags.
  • Class Selector (.classname): Applies styles to elements with a given class.
  • ID Selector (#idname): Targets a unique element using an ID.

2. Advanced Selectors

  • Attribute Selector ([type="text"]): Selects elements based on attributes.
  • Group Selector (h1, p, div): Styles multiple elements at once.
  • Descendant (div p): Targets elements inside a parent element.
  • Child (div > p): Styles only direct children of a parent element.
  • Sibling (h1 + p): Targets elements next to a specific element.
Example 1: * { color: red; } /* Universal Selector */
                h1 { color: blue; } /* Element Selector */
                .classname { color: green; } /* Class Selector */
                #idname { color: purple; } /* ID Selector */
            
Example 2: [type="text"] { background-color: yellow; } /* Attribute Selector */
h1, p, div { margin: 10px; } /* Group Selector */
div p { color: orange; } /* Descendant Selector */
div > p { font-weight: bold; } /* Child Selector */
h1 + p { text-decoration: underline; } /* Sibling Selector */

Colors and Backgrounds

CSS allows customization of colors and backgrounds using different properties.

1. Color Formats

  • Named Colors: red, blue, green.
  • HEX Codes: #ff5733.
  • RGB Values: rgb(255, 87, 51).
  • HSL: hsl(12, 100%, 50%).

2. Background Properties

  • background-color: Sets background color.
  • background-image: Adds background images.
  • background-size: Defines image dimensions.
  • background-position: Controls image placement.
  • background-repeat: Determines if an image repeats.
Example: body {
                background-color: #ff5733; /* HEX */
                background-image: url('image.jpg');
                background-size: cover;
                background-position: center;
                background-repeat: no-repeat;
            }

Fonts and Text Styling

Typography enhances readability and aesthetics.

1. Font Styling

  • font-family: Defines fonts (Arial, serif).
  • font-size: Sets text size (16px, 1.2em).
  • font-weight: Adjusts boldness (bold, 400).
  • font-style: Controls italicization (italic, normal).

2. Text Formatting

  • text-align: Aligns text (left, right, center).
  • line-height: Defines spacing between lines.
  • letter-spacing: Adjusts spacing between characters.
  • text-transform: Converts text to uppercase, lowercase, or capitalized.
Example: p {
                font-family: Arial, sans-serif;
                font-size: 16px;
                font-weight: bold;
                text-align: center;
                line-height: 1.5;
                letter-spacing: 2px;
                text-transform: uppercase;
            }

Box Model

The box model consists of four parts:

  • Content: The actual text or image inside an element.
  • Padding: Space between content and border.
  • Border: The outline surrounding an element.
  • Margin: Space outside the border that separates elements.
Example: div {
                width: 200px;
                height: 100px;
                padding: 10px;
                border: 5px solid black;
                margin: 20px;
            }

Display and Visibility

CSS controls how elements appear and behave on a page.

1. Display Types

  • block: Takes full width, appearing on a new line.
  • inline: Stays within the same line.
  • inline-block: Allows inline elements with width and height.
  • none: Hides an element completely.

2. Visibility

  • visibility: hidden; Thides an element but retains space.
Example: .block { display: block; }
                .inline { display: inline; }
                .inline-block { display: inline-block; }
                .hidden { display: none; }
                .invisible { visibility: hidden; }

Positioning and Layouts

CSS provides different positioning methods for layout control.

1. Positioning

  • static: Default flow positioning.
  • relative: Moves element relative to its normal position.
  • absolute: Positions relative to the nearest positioned ancestor.
  • fixed: Stays in place even while scrolling.
  • sticky: Acts as relative until it reaches a scroll threshold.

2. Flexbox

  • A one-dimensional layout model for aligning elements horizontally or vertically.

3. CSS Grid

  • A two-dimensional system for arranging elements in rows and columns.
Example 1: div {
                position: absolute;
                top: 50px;
                left: 100px;
            }

Example 2: .container {
    display: flex;
    justify-content: center;
    align-items: center;
} 

Example 3: .grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
}

CSS Transitions and Animations

CSS enables dynamic visual effects

1. Transitions

  • Adds smooth effects when a property changes.
  • Uses transition-property, transition-duration, and transition-timing-function.

2. Animations

  • Uses @keyframes to define motion effects.
  • Properties like animation-name, animation-duration, and animation-delay control animation behavior.
Example 1:div {
                transition: background-color 0.5s ease;
            }
            
            div:hover {
                background-color: red;
            }

Example 2: @keyframes example {
    from {background-color: red;}
    to {background-color: yellow;}
}

div {
    animation-name: example;
    animation-duration: 4s;
}

Responsive Web Design

Ensures web pages adapt to different screen sizes.

1. Media Queries

  • Uses @media to apply styles based on device width.

Viewport Meta Tag

  • (meta name="viewport" content="width=device-width, initial-scale=1.0") ensures mobile responsiveness.
Example: @media (max-width: 600px) {
                body {
                    background-color: lightblue;
                }
            }

CSS Variables

CSS variables store reusable values to improve consistency.

Key Concepts:

  • Declared using --var-name: value; inside :root.
  • Used with var(--var-name).
Example: :root {
                --main-color: blue;
            }
            
            div {
                color: var(--main-color);
            }

2. Pseudo-Elements

Example: a:hover { color: red; } /* Pseudo-Class */

                p::first-letter { font-size: 2em; } /* Pseudo-Element */

Shadows and Effects

Greedy algorithms make locally optimal choices at each step to find a global optimum.

1. Box Shadow

  • Adds shadow effects to elements using box-shadow.

2. Text Shadow

  • Creates shadow effects on text with text-shadow.

3. Opacity and Filters

  • opacity: Controls element transparency.
  • filter: Applies effects like blur and brightness.
Example: div {
                box-shadow: 5px 5px 10px gray;
                text-shadow: 2px 2px 5px black;
                opacity: 0.8;
            }

Advanced CSS Features

1. CSS Grid Layout

  • Defines rows and columns for advanced layouts.

2. Flexbox

  • Aligns elements efficiently in a flexible container.

3. CSS Clipping & Masking

  • clip-path: Clips elements into custom shapes.
  • mask: Hides or reveals portions of an element.

4. CSS Blend Modes

  • mix-blend-mode: Blends elements with the background.
Example: .clip-example {
                clip-path: circle(50%);
            }
            
            .blend-mode {
                mix-blend-mode: multiply;
            }

CSS Best Practices

  • Use external stylesheets for scalability
  • Follow a mobile-first approach.
  • Use CSS variables for maintainability
  • Minimize CSS for faster load times.
